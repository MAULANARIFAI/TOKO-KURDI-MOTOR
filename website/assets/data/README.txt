You can manage products via CSV exported from Excel.

- Put your CSV at: assets/data/products.csv
- Use the header:
  id,name_id,name_en,brand,sku,compat_id,compat_en,price,img,badge
- See products-template.csv for examples.

Notes:
- price is number in IDR (no thousands separator). You can leave blank.
- badge: original or aftermarket
- img: relative path like assets/img/products/oil-filter.jpg
